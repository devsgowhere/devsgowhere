**Below is the schedule,**

🧱 **10.30am - 11.10am : Topic - Build a Tetris game in 4 minute using Spec driven development**

In this session, we explore how anyone can build functional software in minutes using AI — not by learning to code, but by learning to think and write clearly. Using a live demonstration of building a fully playable Tetris game with an AI autoplay feature from a single structured prompt, we unpack why the quality of your specification is the true bottleneck in AI-assisted development. The same idea, described two different ways, can produce wildly different results — one that works straight away, and one that falls apart.

Move beyond the "hobbyist" applications of Generative AI to witness its power in solving high-stakes, engineering challenges. Join us to see how human judgment and AI execution unite to solve a real-world problem to set up better computing infrastructure.
